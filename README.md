# WPQuiz

Aplikacja umożliwia rozwiązywanie niebanalnych testów udostępnionych przez platformę Wirtualna Polska. Sprawdź swoją więdzę w takich katergoriach jak geografia, kino, historia i dzisiątki innych. Twój wynik nie jest satysfakcjonujący? Spróbuj go poprawić, tym razem na pewno się uda!

![1](https://user-images.githubusercontent.com/17621860/74863527-53667700-534e-11ea-9c35-a8997aa2c24e.jpg)
![2](https://user-images.githubusercontent.com/17621860/74863528-53ff0d80-534e-11ea-9656-ce2074027d39.jpg)

Podczas wypełniania testu zawsze widzisz na jakim jesteś etapie, dzieki paskowi progresu. Jeśli nie masz teraz wystarczająco czasu, aby dokończyć test, możesz to zrobić poźniej.
